# Simple-Todo-App-Flask

- Build & Export React section  
  - Change to **editor** directory  
  ``` 
  cd editor 
  ```
  - Import modules
  ```
  yarn install
  ```
  - Build and export
  ``` 
  yarn run build
  ```  

- Install python modules
```
pip install -r requirements.txt
```
- Run with Flask
```
python3 main.py
```
